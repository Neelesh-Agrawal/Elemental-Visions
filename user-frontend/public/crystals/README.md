# Crystal Images Directory Structure

This directory contains all the crystal images for the Elemental Visions website.

## 📁 Folder Organization

Each crystal has its own folder with specific image files for different forms.

### 🔮 Individual Crystals (Need all forms)

#### Standard Crystal Forms (5 images each):
- **bracelet.jpeg** - Crystal in bracelet form
- **pendant.jpeg** - Crystal as pendant
- **tumble.jpeg** - Tumbled stones (set of 2)
- **tree.jpeg** - Crystal tree/bonsai form
- **raw.jpeg** - Raw/natural crystal form
- **outer.jpeg** - Main crystal image (for card display)

## 📝 Image Guidelines

### ✅ **Best Practices:**
- **Format**: JPEG files only (all images should be saved as .jpeg)
- **Size**: Minimum 800x800px (square preferred)
- **Quality**: High resolution, clear and bright
- **Background**: Clean, neutral background preferred
- **Naming**: Exact file names as shown above (case-sensitive)
- **Extension**: ALL images must use .jpeg extension

### 🎯 **For Each Crystal Form:**
- **Bracelet**: Show the crystal beads in bracelet form
- **Pendant**: Single crystal pendant/charm
- **Tumble**: 2 small tumbled stones
- **Tree**: Crystal tree/bonsai decoration
- **Raw**: Natural, uncut crystal specimen
- **Outer**: Best representative image of the crystal

---

## 🚀 **How to Add Images**

1. Navigate to the specific crystal folder
2. Add your images with the exact file names shown
3. Refresh the website - images will appear automatically!
4. No code changes needed - just add the image files
